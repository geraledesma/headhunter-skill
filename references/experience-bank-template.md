# Experience Bank

This file is the **single source of truth** for all CV content, cover letters, and outreach
framing. headhunter will never add content that isn't in this file. If something is missing,
it's marked `[MISSING]` in outputs.

Rules:
- Only include facts you can verify and speak to in an interview
- Use specific numbers, percentages, and dollar amounts wherever possible
- Note the exact scope of each claim (e.g., "part of a 3-person team" not "led a team")
- Run an Experience Bank Session to fill this out with headhunter's guided Q&A

## Framing hierarchy

When the same experience can be framed in multiple ways, use this priority order:

1. **Quantified outcome** — a specific metric is always strongest
2. **Scope indicator** — scale, complexity, or stakeholder seniority
3. **Action verb accuracy** — match the verb to the actual level of ownership
   (managed ≠ led ≠ supported ≠ contributed to)
